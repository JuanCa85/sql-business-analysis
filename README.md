# Análisis de datos con SQL

## Descripción del proyecto

Este proyecto consiste en la extracción y análisis de datos desde una base relacional utilizando consultas SQL para responder preguntas de negocio.
Se realizaron joins, agregaciones y filtros para generar insights accionables.

**Objetivo**

Responder preguntas de negocio mediante consultas SQL sobre una base de datos relacional.

**Herramientas utilizadas**

- SQL
- PostgreSQL
- Jupyter Notebook

**Proceso del análisis**

1. Exploración de la base de datos.
2. Consultas con filtros y condiciones.
3. Joins entre tablas.
4. Agregaciones y métricas.
5. Generación de conclusiones.

**Resultados principales**

- Respuestas a preguntas clave de negocio.
- Identificación de patrones en los datos.

**Habilidades demostradas**

 - Consultas SQL
 - Joins y subconsultas
 - Agregaciones
 - Análisis de datos estructurados
